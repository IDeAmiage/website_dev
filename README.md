## Installation

Install node: [here](https://nodejs.org/en/download/)

Verify your installation:
`npm --version`

Install angular: `npm install -g @angular/cli`

### Run the project:
 `cd IDeA` </br>
 `ng serve`

Access to localhost: [here](http://localhost:4200/)
